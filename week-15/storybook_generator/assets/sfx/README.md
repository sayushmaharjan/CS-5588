# SFX Assets

Drop `.wav` files here with the exact filenames expected by `SFX_MAP` in `sound_engine.py`:

| File | Trigger keyword |
|------|----------------|
| `forest_ambience.wav` | `forest` |
| `sparkle.wav` | `magic`, `sparkle` |
| `crickets.wav` | `night` |
| `waves.wav` | `ocean` |
| `wind.wav` | `wind` |
| `rain.wav` | `rain` |
| `birds.wav` | `birds` |
| `fireplace.wav` | `cozy` |
| `stars_chime.wav` | `stars` |

**Sources:** Free audio from [freesound.org](https://freesound.org), [pixabay.com/music](https://pixabay.com/music/), or [zapsplat.com](https://www.zapsplat.com).

If a file is missing, the SFX layer is silently skipped for that page — the narration and music still play normally.
